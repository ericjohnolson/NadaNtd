What is an administrative level?

An administrative level is a subnational entity or geographic division within a country. Generally the government establishes regions, provinces, districts and other types of administrative levels. In this tool, the administrative levels included will relate specifically to data collection in your NTD program.

Which administrative levels are included in this tool?

The administrative levels used in this database are specific to program implementation and data collection in your country. If NTD programs store data from individual villages, they may choose to include villages as the smallest administrative level in this tool. Other NTD programs may find the smallest administrative level needed is Primary Health Units (PHU), Community Health Facilities, or another unit that groups villages. For other NTD programs, the smallest level of data collection may be city, district, or wherever lower level data is collated.

How many administrative levels can I include in this tool?

The tool requires at least two administrative levels and allows no more than seven levels.

 

How do I add a sentinel site?

If you are entering data for a sentinel survey and the sentinel site is not included in the Site Name drop-down list, add a sentinel site.

1.     On the Sentinel/Spot Check Site Survey screen, select Add new site.

2.     Select the location of the sentinel site.

3.     Enter the site name.

4.     Enter the latitude and longitude of the site. (Optional.)

5.     Enter any notes on this sentinel site for your reference.

 

 

How do I add, edit, or delete partners or funders?

On the Sentinel/Spot Check Site Survey screen:

1.     To add: Select Add new Funder/Partner on any survey or intervention form. Enter the name of the new partner or funder and click Save.

2.     To edit: select the name of the Funder/Partner you want to change.

3.     To delete: select the name of the Funder/Partner you want to remove.

 

How do I edit or delete the names of partners or funders?

1.     On the Sentinel/Spot Check Site Survey screen, select the name of the Funder or Partner you want to change.

2.     Select Edit or Delete to make the desired changes.

 

 

How do I add custom indicators to a form?

You can add new indicators to any form. This allows you to match the National Database Template to your programs data collection needs over time. After you create a custom indicator, it will appear on the form from then on.

1.     On the screen [wherever it ends up; be specific], select Add/remove indicators [image]

2.     Select Add New

3.     Enter:

  Indicator name

  Data type [link]

  Required field(s) [tbd]

4.     Save the indicator.

5.     Review the indicator you just created and Edit if necessary.

6.     Save when finished. Your new indicator will be listed at the bottom of the form from then on.

General remarks:

  While you will be able to add data for the new indicator from now on, there will be no data for the custom indicator for any previously entered forms. You can view and edit old forms to add data for the new indicator, if necessary.

  Custom indicators will not be included on the Joint Reporting Form. You will need to add them manually by manipulating the data in Excel. See: How to export data to Excel

 

How do I import data?

Alternate title: Create and upload an import file

See also:

Importing data is one way to enter data into the National Database Template. Importing data can be especially useful if your data is already in Excel because you will not need to enter information into the National Database Template one screen at a time. You will be guided to generate special import Excel worksheets from the database to use this function.

1.     On the form [location tbd], select Create import file [image]

2.     Use the Level of implementation drop down menu to indicate the administrative level of data you plan to import.

3.     Select the locations you want data for by using the arrow keys and/or double clicking. [image]

4.     Create the import file, and save the resulting Excel file on your computer.

5.     Open the spreadsheet. The row and column headers will be populated with the names of selected locations and the indicators.

6.     Enter your data into the Excel file. When finished, close the excel file, and return to the National Database Template.

7.     Select Upload import file. A message box will notify when your data has been successfully imported. [Image]

 

 

What is a data type?

See also: add a new indicator; data validation]

A data type is a data format that contains a specific type of values, such as number, percent, text, yes/no, and date. Most indicators can only be entered in the data type specified for it. When you add a custom indicator, you need to specify its data type.

 

 

How do I export my data to Excel?

Alternate title: Creating reports

See also:

Text:

Export data to generate an Excel spreadsheet ready for data analysis. You can customize the export file by choosing the indicators and administrative levels necessary to complete your task.

 

1.     On the Reports tab, select View Reports [update location if needed]
